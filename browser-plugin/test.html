JMiMi AI Assistant Test Page

This page is designed to test the JMiMi browser extension functionality.

Testing Instructions:

1. Make sure the JMiMi extension is loaded in your browser
2. You should see a blue floating button on the right side of the page
3. Click the floating button to toggle the main UI panel
4. The panel should slide in from the right when opened and slide out when closed
5. You can drag the floating button to reposition it

If you don't see the floating button, check your browser's developer console for any error messages.
